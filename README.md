# IT-Home-Lab-Portfolio

## Overview
A two-part IT home lab project built to simulate a real corporate IT environment.

## Lab 1 — Help Desk Ticketing System
- Deployed osTicket on a Windows 10 VM using XAMPP (Apache, MySQL, PHP)
- Configured SLA plans, departments, agents, and help topics
- Resolved 8+ simulated tickets covering password resets, hardware issues, and a business critical outage escalation

## Lab 2 — Active Directory Administration
- Built a Windows Server 2022 domain controller using Oracle VirtualBox
- Installed Active Directory Domain Services and created the helpdesk.lab domain
- Created organizational units, domain user accounts, and security groups
- Performed real AD tasks: account unlock, password reset, employee offboarding, and new hire provisioning
- Joined a Windows 10 workstation to the domain

## Technologies Used
VirtualBox, Windows 10, Windows Server 2022, osTicket, XAMPP, Active Directory, DNS, PowerShell
